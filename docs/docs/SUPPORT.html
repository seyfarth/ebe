EBE on GitHub

EBE, Easy Beginner Environment, is an integrated development environment supporting assembly language using the nasm assembler and also C/C++ using g++ (or clang acting like g++). The intended purpose is to make it easier for beginners to learn assembly. It runs on Windows, OS X, Linux and probably could run on most UNIX-like systems.
